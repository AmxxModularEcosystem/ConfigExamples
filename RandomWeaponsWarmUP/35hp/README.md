# Разминка на ножах с 35hp

Обычная разминка 35hp. В `Modes.json` указана только разминка на ножах, можно дописать её к вашему набору режимов.

Требуются следующие плагины:

- [RandomWeaponsWarmUP](https://github.com/ArKaNeMaN/amxx-RandomWeaponsWarmUP)
- [ItemsController](https://github.com/AmxxModularEcosystem/VipModular)
