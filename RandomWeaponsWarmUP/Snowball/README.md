# Добавление режима разминки на снежках к обычным режимам

В зимнее время можно добавлять в привычный пул разминок снежную, которая будет запускаться с некоторым шансом. Либо можно оставить в файле `Modes.json` только снежную разминку, если нужно чтобы запускалась только она.

Требуются следующие плагины:

- [RandomWeaponsWarmUP](https://github.com/ArKaNeMaN/amxx-RandomWeaponsWarmUP)
- [ItemsController](https://github.com/AmxxModularEcosystem/VipModular)
- [IC-I-Cwapi](https://github.com/AmxxModularEcosystem/IC-I-Cwapi)
- [CustomWeaponsAPI](https://github.com/AmxxModularEcosystem/CustomWeaponsAPI)
- [CWAPI-A-Snowball](https://github.com/AmxxModularEcosystem/CWAPI-A-Snowball)
